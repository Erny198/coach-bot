# 🌟 Coaching Deep Transformation Bot

Telegram-бот для практики коучинга глубинной трансформации.
---

## 🎯 Три режима работы

### 🧑‍💼 Режим «Коуч» (бот коучит пользователя)
Бот проводит полноценную коучинговую сессию по фреймворкам:
- **SCORE** — системное исследование проблемного пространства
- **Хорошо Сформулированный Результат (ХСР)** — постановка намерения
- **Стратегии Гениев** — создание образа будущего (Р. Дилтс)
- **Квадрат Декарта** — исследование мотивации
- **Позиции Восприятия** — 4 точки зрения
- **Нейрологические Уровни** — уровни опыта

### 🎓 Режим «Коучи» (бот = клиент-executive)
Бот играет роль реалистичного клиента — executive/собственника бизнеса.
6 профилей с типичными запросами:
- **Выгорание CEO** — 15 лет строил бизнес, «бегу на пустом баке»
- **Потеря доверия** — партнёр предал, не может доверять
- **Масштабирование и страх** — стартап растёт, а основатель чувствует себя самозванцем
- **Поиск смысла** — всё есть, а смысла нет
- **Кризис лидерства** — ключевые люди уходят, одиночество на вершине
- **Семья vs бизнес** — дочь сказала «ты мне чужая»

Каждые 4-5 обменов бот даёт **обратную связь тренера** (оценка из 10).

### 📚 Режим «Тренер» (тренировка вопросов по темам)
Бот даёт мини-ситуации и тренирует задавать правильные коучинговые вопросы.

**13 тем для тренировки:**

| # | Тема |
|---|------|
| 1 | 🧭 Центровка и качество действий: «Компас» для коучей |
| 2 | 🔮 Многоуровневое Восприятие: коуч, клиент и поле между ними |
| 3 | 🖐 Модель Уточнения «Пальчики» (Мета-Модель) |
| 4 | ⏪ Искусство Отмотки и нейробиология |
| 5 | 🎯 Постановка Персонального Намерения |
| 6 | 🧘 Медитация «Установка намерения на поддержку научения» |
| 7 | 👁 Паттерны Внимания: 12 режимов |
| 8 | 🌟 Прояснение хорошо сформулированного намерения |
| 9 | 📐 Структура и согласование ХСР |
| 10 | 🪑 Модель «2 стула» контрастного анализа |
| 11 | 💬 Эффективные речевые паттерны |
| 12 | 🎯 SCORE |
| 13 | 🚀 Стратегии Гениев |

## 🚀 Быстрый старт

### 1. Получите токены

**Telegram:** @BotFather → `/newbot` → скопируйте токен

**Anthropic:** [console.anthropic.com](https://console.anthropic.com) → создайте API key

### 2. Установите зависимости

```bash
pip install -r requirements.txt
```

### 3. Запустите

```bash
export TELEGRAM_TOKEN="ваш_токен"
export ANTHROPIC_API_KEY="ваш_ключ"
python bot.py
```
